## Dcat

Dcat is a 3d rendering engine that works entirely within the terminal. It supports almost every major 3d file format and allows viewing 3d files in terminal environments, such as SSH.

### Features
- Animation support
- Kitty Graphics Protocol support
- Sixel support
- Terminal resize detection
